這個資料夾只有「在 repo 根目錄直接 docker compose up（開發者測試 image 時）」會用到。

一般使用者**不會**用到這個資料夾。使用方式是：
  cd 到放有 xxx.ovpn 的任意目錄，執行 install 指令即可。
  install 腳本會把該目錄當作 /vpn:ro mount 進容器，自動抓 *.ovpn。

開發者本地測試 image：把任意 *.ovpn 放在這裡（會被 .gitignore 忽略），
然後從 repo 根目錄執行：
  docker compose build
  docker compose up -d
  docker compose exec -u student lab bash
